# 实验室生产数据统计看板（PyQt6）

## 1. 功能覆盖
应用通过外链读取生产数据 `xlsx/xlsm`，不改动原表 `A~AI` 已有内容。

已实现需求：
1. 筛选明细：新增“筛选明细”Tab，列出筛选命中的相关条目用于核查。
2. 在研负载：按实验员统计未完成分子数（`AG` 为空，`Unique(C|D)`）并给出“可接新任务/已饱和”。
3. 实验员积分：按指定时间段 + 可选筛选（实验员、WBP Code、Product ID）统计每位实验员积分。
4. 偶联类型积分：按时间段统计全体记录中 `IC Complete` 与 `Other` 的总积分与占比。
5. FTE 总览：`FTE = IC Complete积分 / 总积分`，支持实验员筛选。
6. 按实验员 FTE：给定时间段列出每位实验员 FTE，并计算 `FTE总值(Σ个人FTE)`。
7. 人效：`Unique(C|D)数量 / FTE总值`。
8. 右侧统一图表区：随当前 Tab 自动切换图表（柱状/环图/趋势）。

## 2. 字段映射（固定）
- `A`：偶联类型（仅区分 `IC Complete` 和其他）
- `C`：WBP Code
- `D`：Product ID
- `E`：实验员
- `AF`：起始日期
- `AG`：完成日期（空=未完成）
- `AI`：实验积分（score）
- 数据起始行：第 4 行（1~3 行为 header）

## 3. 时间段与日期口径
- 时间段：可直接指定“起始日期 + 结束日期”（闭区间）。
- 快捷时间：支持“一键套用本周/本月”，生成后可手动微调起止日期。
- 统计日期口径：
  - 完成优先：优先取 `AG`，若空取 `AF`
  - 完成日期：仅 `AG`
  - 起始日期：仅 `AF`
